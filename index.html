CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • In the command line, pull down the latest main branch by entering git pull origin main
  • Use the following Git command to create a new branch named feature/add-html. git checkout -b feature/add-html
  • it's important to frequently sync the local version of your code with the remote version in GitHub. This matters, especially when working on a team, so that a functional, updated version of the project can be shared and accessed by all team members
